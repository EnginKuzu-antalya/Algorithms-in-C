# Insertion Sort

Bu proje, temel sıralama algoritmalarından biri olan Insertion Sort  algoritmasının C dili ile basit bir uygulamasını içerir. Algoritma, her adımda dizinin bir sonraki elemanını alıp, önceki sıralanmış kısımda uygun yere yerleştirerek çalışır.

Özellikler
Dizinin her iterasyondaki (adımındaki) durumunu ekrana yazdırır.

Dinamik olarak dizi boyutunu hesaplar.

Küçükten büyüğe sıralama yapar.
